# Composite Camera Texture

## What is this about?

The Composite Camera Texture package provides tools for combining and compositing multiple camera feeds and textures in Lens Studio. It enables advanced camera manipulation, blending, and layering for complex AR experiences.

## How to use

1. **Import the package** into your Lens Studio project
2. **Add the prefab** to your scene from the Prefabs folder
3. **Configure camera sources** and composite settings
4. **Adjust blending modes** and layer properties as needed

## Details

- **Camera Compositing**: Combine multiple camera feeds with different blend modes
- **Real-time Processing**: Efficient real-time texture composition
- **Custom Shaders**: Advanced shader-based compositing operations
- **Layer Management**: Organize and control multiple texture layers

## Description of Scripts

- **CameraService.ts**: Manages camera input and texture handling
- **CompositeImage.prefab**: Main prefab for camera compositing setup
- **Custom Shaders**: Various shader files for different compositing effects
