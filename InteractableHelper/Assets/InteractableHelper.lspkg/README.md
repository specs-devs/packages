# Interactable Helper

## What is this about?

The Interactable Helper package provides utilities and components for creating interactive objects in Lens Studio. It simplifies the process of making objects respond to user input, gestures, and interactions.

## How to use

1. **Import the package** into your Lens Studio project
2. **Add the InteractableHelper script** to objects you want to make interactive
3. **Configure interaction settings** in the inspector
4. **Set up event handlers** for different interaction types

## Details

- **Gesture Recognition**: Built-in support for tap, drag, and other gestures
- **Event System**: Comprehensive event handling for user interactions
- **Visual Feedback**: Easy-to-implement visual responses to interactions
- **Performance Optimized**: Efficient interaction detection and processing

## Description of Scripts

- **InteractableHelper.ts**: Main script for making objects interactive
- **Examples/**: Sample scenes and prefabs demonstrating various interaction patterns
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 

## Package Configuration

This package has been configured with proper visibility settings (CurrentVisibility: Locked, MaxVisibility: Editable) for optimal distribution.
