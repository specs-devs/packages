# Marker Tracking Helper

## What is this about?

The Marker Tracking Helper package provides utilities and components for image marker tracking in Lens Studio. It simplifies the process of setting up and managing image marker detection and tracking.

## How to use

1. **Import the package** into your Lens Studio project
2. **Add image markers** to your scene using the provided marker files
3. **Configure tracking settings** and marker properties
4. **Set up tracking events** and responses

## Details

- **Image Marker Tracking**: Detect and track predefined image markers
- **Easy Setup**: Simple marker configuration and management
- **Event System**: Comprehensive event handling for marker detection
- **Performance Optimized**: Efficient marker detection and tracking

## Description of Scripts

- **Scripts/**: Marker tracking and management scripts
- **Examples/**: Sample scenes demonstrating marker tracking
- **Materials/**: Custom materials for marker visualization
- **Meshes/**: 3D models for marker-related elements
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 

## Package Configuration

This package has been configured with proper visibility settings (CurrentVisibility: Locked, MaxVisibility: Editable) for optimal distribution.
