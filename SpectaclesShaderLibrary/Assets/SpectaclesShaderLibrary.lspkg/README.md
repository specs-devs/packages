# Spectacles Shader Library

A collection of seven optimized shaders designed to enhance power efficiency without sacrificing visual quality. These shaders provide a better balance between performance and aesthetics, perfect for Spectacles applications.

## Overview

The Spectacles Shader Library focuses on power-efficient rendering while maintaining high visual quality. All shaders are optimized for Spectacles hardware and can be used as drop-in replacements for traditional shaders.

## Available Shaders

| Shader | Description |
|--------|-------------|
| **Uber Specs** | Full-featured shader with comprehensive material controls |
| **Unlit Specs** | Simple unlit shader for maximum performance |
| **Matcap Specs** | Material capture shader for stylized reflections |
| **Simple Specs** | Lightweight shader with basic lighting |
| **Toon Specs** | Cel-shaded toon shader for stylized looks |
| **Rim Highlight Specs** | Shader with rim lighting effects |
| **Outer Glow Specs** | Shader with outer glow/outline effects |

## How to Use

### Using Materials

1. Navigate to the `Materials/` folder
2. Select the shader material you want to use
3. Apply it to your mesh or object
4. Customize the material properties in the Inspector

### Examples

Check the `EXAMPLES/` folder for:
- Pre-configured example materials
- Sample prefabs demonstrating each shader
- Reference textures and assets
