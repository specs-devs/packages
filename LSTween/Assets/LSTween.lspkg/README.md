# LS Tween

## What is this about?

The LS Tween package provides animation and tweening capabilities for Lens Studio. It enables smooth transitions, animations, and interpolations for objects, properties, and values over time.

## How to use

1. **Import the package** into your Lens Studio project
2. **Add the LSTween script** to objects you want to animate
3. **Configure tween parameters** like duration, easing, and target values
4. **Start animations** using the provided tween methods

## Details

- **Property Tweening**: Animate position, rotation, scale, and other properties
- **Easing Functions**: Multiple easing curves for natural motion
- **JavaScript & TypeScript**: Available in both JS and TS versions
- **Performance Optimized**: Efficient animation system with minimal overhead

## Description of Scripts

- **LSTween.ts**: Main tweening engine and API
- **RotationInterpolationType.ts**: Defines rotation interpolation methods
- **TweenJS/**: JavaScript implementation examples
- **Examples/**: Sample scenes demonstrating various tweening techniques
