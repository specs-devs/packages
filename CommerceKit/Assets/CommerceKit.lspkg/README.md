# CommerceKit (Closed Beta)

CommerceKit is a framework that enables you to handle in-app purchases for non-consumable items in Spectacles. With CommerceKit, you can define product information, facilitate payments, deliver in-app purchases and manage user purchases.

## Documentation and API Reference

- **Getting Started**: Learn how to use CommerceKit by following the [Get Started Guide](https://developers.snap.com/spectacles/spectacles-frameworks/commerce-kit/getting-started).

## How to Set Up CommerceKit in Your Project

Follow these steps to configure CommerceKit in your project:

1. Ensure the Spectacles Interaction Kit Prefab is added to your scene.
2. Add the ProductCatalog script file to your Scene.
3. Configure your ProductCatalog with the desired in-app purchases.
4. Manage your products through the CommerceKit.ts script.
5. [Optional] Drag and drop the CommerceKit Example to see an example.

You're now ready to start building with CommerceKit!
